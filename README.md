Необходимое ПО: nodejs v8.4.0, npm 5.30


Инструкция по запуску:

0. Ставим зависимости

в корне проекта запускаем

:~$ npm i

1. Запускаем сервер

переходим из корня проекта в папку сервер запускаем

:~$ node index.js

2. Собираем приложение

в корне проекта запускаем:

:~$ npm run dev
